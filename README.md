##  Sobre mí

Soy Analista de Datos con enfoque estratégico, visual y humano. Me especializo en transformar información compleja en soluciones accesibles, cálidas y accionables. Utilizo herramientas como **Power BI**, **SQL**, **Python** y tecnologías web para crear dashboards interactivos, modelos relacionales y automatizaciones que impulsan decisiones inteligentes.
